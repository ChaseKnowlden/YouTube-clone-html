## 📖 About The Project

This project is a pixel-perfect clone of YouTube's user interface, showcasing modern web development practices using only vanilla HTML, CSS, and JavaScript. It demonstrates responsive design principles, interactive UI components, and clean code architecture without relying on any frameworks or libraries.

Perfect for learning web development fundamentals or as a starting point for building your own video platform!

### ✨ Key Highlights

- 🎨 **Pixel-Perfect Design** - Faithful recreation of YouTube's modern interface
- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- 🚀 **Vanilla JavaScript** - No frameworks, no dependencies, just pure JavaScript
- ♿ **Accessible** - Built with accessibility best practices in mind
- 🎯 **Clean Code** - Well-structured, documented, and maintainable codebase

## 🚀 Getting Started

### Prerequisites

All you need is a modern web browser! No installations, no dependencies, no build process.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ChaseKnowlden/YouTube-clone-html.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd YouTube-clone-html
   ```

3. **Open in your browser**
   ```bash
   # Option 1: Double-click index.html
   
   # Option 2: Use a local server (recommended)
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (if you have npx)
   npx serve
   
   # PHP
   php -S localhost:8000
   ```

4. **Open your browser**
   ```
   Navigate to http://localhost:8000
   ```

That's it! The application should now be running in your browser. 🎉

## 💡 Usage

### Basic Navigation

- **Hamburger Menu** - Click to toggle the sidebar
- **Search Bar** - Enter search terms (UI only, no backend integration)
- **Voice Search** - Click the microphone icon for voice search UI
- **Sidebar Items** - Click navigation items (UI demonstration)

### Responsive Behavior

- The layout automatically adapts to your screen size
- On mobile devices, the sidebar becomes a slide-out menu
- Touch gestures work smoothly on mobile devices

### Customization

You can easily customize the project:

1. **Colors** - Modify CSS variables in `styles.css`
2. **Layout** - Adjust flexbox properties and breakpoints
3. **Content** - Add your own video thumbnails and content
4. **Functionality** - Extend JavaScript features in `script.js`

## 📁 Project Structure

```
YouTube-clone-html/
│
├── index.html          # Main HTML file with semantic markup
├── styles.css          # All CSS styles and responsive design
├── script.js           # JavaScript for interactive functionality
├── .gitignore          # Git ignore file
└── README.md           # Project documentation
```

### File Descriptions

- **`index.html`** - Contains the semantic HTML structure including navbar, sidebar, and main content area
- **`styles.css`** - All styling including layout (Flexbox), colors, typography, and responsive breakpoints
- **`script.js`** - Handles sidebar toggle, click-outside functionality, and other interactive features
